## Getting Started

### Prerequisites

Before running Marqle on your local machine, ensure you have the following installed:

- [Node.js](https://nodejs.org/) (v12.18.3 or higher)
- [npm](https://www.npmjs.com/) (v6.14.6 or higher)

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/marqle.git
   ```

2. Navigate to the project directory:
   ```bash
   cd marqle
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Open the `index.html` file in your preferred web browser to view the landing page locally.

## Usage

Customize Marqle to align with your digital marketing agency's brand and services:

- Modify the content in the HTML files to add your services, team information, and contact details.

## Deployment

To deploy Marqle on a live web server, follow these steps:

1. Purchase a domain name (e.g., www.yourdigitalagency.com) and set up web hosting.

2. Upload all project files to your web hosting server using FTP or a hosting control panel.

3. Configure your web server to serve the `index.html` file as the default page.

4. Ensure that your domain name's DNS settings point to your web hosting server.
